Sample model reference

This section presents BornAgain simulations of various model systems. All Python scripts are shipped with BornAgain. They can be found in the “Examples” directory:

  • Windows
  • MacOS
  • Linux 

C:\BornAgain-1.11.1\Examples\python

/Applications/BornAgain.app/Contents/share/BornAgain-1.11/Examples/python

install_dir/share/BornAgain-1.11/Examples/python

The examples are subdivided into the following categories:
  • Embedded particles
  • Layered structures
  • Interference functions
  • Complex Shapes
  • Beam and Detector
  • Fitting
  • Miscellaneous
